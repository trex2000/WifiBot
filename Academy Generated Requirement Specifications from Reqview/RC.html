1.3.4.2.1 Inputs
  • A0 – the value of the light sensor
  • D7 – connects the GPIO13 pin of the Wi Fi module to the Arduino (Future Up button)
  • D8 – connects the GPIO12 pin of the Wi Fi module to the Arduino (Future Down button)
  • D12– connects the GPIO4 pin of the Wi Fi module to the Arduino (Future Left button)
  • D13– connects the GPIO5 pin of the Wi Fi module to the Arduino (Future Right button) 

The power needed for powering the module will be provided from the DC – DC module output
46
1.3.4.2.2 Outputs
  • 5V – power for the Wi – Fi Conversion module which will privide 3.3V for the pins: VCC, CH_PD and GPIO2
  • GND – ground reference for the Wi – Fi module pins: GPIO0, GPIO15 and GND
  • Analog pin 1 – connects with the RY pin on the Light Module,but it has to be set as a DIGITAL PIN
  • Analog pin 2 – connects with the LY pin on the Light Module,but it has to be set as a DIGITAL PIN
  • Digital pin 2 – connects with the R pin on the Light Module
  • Digital pin 3 – connects with the ENB pin on the Hbridge Module, this pin needs to be a PWM pin.
  • Digital pin 4 – connects with the Hbridge pin IN4, is not necessarly to be a pwm pin.
  • Digital pin 5 – connects with the Hbridge pin IN3, is not necessarly to be a pwm pin.
  • Digital pin 6 – connects with the W pin on the Light Module
  • Digital pin 9 – connects with the Hbridge pin IN2, is not necessarly to be a pwm pin.
  • Digital pin 10 – connects with the Hbridge pin IN1, is not necessarly to be a pwm pin.
  • Digital pin 11 – connects with the ENA pin on the Hbridge Module, this pin needs to be a PWM pin.
